# Tools and Resources

- Google Drive: penyimpanan dan sharing dokumen. Fetch kadang return kosong di percobaan pertama, retry call yang sama biasanya berhasil sebelum disimpulkan ga bisa diakses. Search fullText contains dengan OR di beberapa term (misal "study group", "bootcamp", "Delta V") efektif buat nemu dokumen program by content, bukan cuma by title.
- Google Docs: sharing eksternal, Commenter access, bukan Editor, buat stakeholder Monad.
- Canva: bikin slide dan visual asset. Ghoza build deck-nya sendiri, Claude cukup kasih copy siap paste, bukan generate file.
- Remix IDE: development dan deploy smart contract, dipakai dalam konteks workshop atau ngajar (contoh: deploy MockERC20 di Remix VM Osaka).
- Google Forms atau Typeform: feedback collection event. Section standar: profil peserta, overall experience, event execution, technical dan content quality, Monad-specific dev experience (ini kritis, jadi material report ke Monad Foundation), community dan networking, open-ended. Timing: 24 jam post-event, target completion di bawah 5 menit.
- Notion: target format referensi dokumen.
- Excel: tracking peserta atau study group. Ada template reusable buat DevRel lain bikin sheet study group sendiri.
